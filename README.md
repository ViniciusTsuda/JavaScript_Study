# Estudos em JavaScript

Repositório criado com o objetivo de estudar, praticar e consolidar conceitos fundamentais e intermediários de JavaScript, servindo como base para aprendizado contínuo e construção de projetos futuros.

Este repositório reúne exemplos práticos, exercícios e pequenos testes desenvolvidos durante o processo de estudo da linguagem.

OBS: O estudo começa com a Seção 5 por praticidade minha, por compreender internamente que o estudo das seções anteriores já estavam bem sólidas na minha cabeça.

---

## Tecnologias utilizadas
- JavaScript (ES6+)
- Node.js (quando necessário)
- Visual Studio Code

---

## Conteúdos abordados
- Variáveis e tipos de dados
- Operadores e estruturas de controle
- Funções e escopo
- Arrays e objetos
- Métodos de array (map, filter, reduce, etc)
- Funções arrow
- Template strings
- Conceitos de `this`, `prototype` e classes
- Manipulação básica do DOM
- Lógica de programação
- Boas práticas em JavaScript

---

## Estrutura do repositório
Os arquivos estão organizados de forma simples, separando exemplos e exercícios conforme os temas estudados.

Exemplo:
- fundamentos/
- funcoes/
- arrays/
- objetos/
- exercicios/
